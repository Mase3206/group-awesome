A Guide To The Galaxy

For our app we wanted to create a sort of guide for different space tours. We broke it up into two main apps for the tours, planetary and stellar, that users could easily navigate to and see the available tours. Each one has a following page dedicated to information about that specific tour. Our database stores all the information regarding each tour making it easy to add or remove tours.

Features Overview

  • Tour Apps: Information about the tours including a description, location, number of people allowed and more. Each section has a separate field in the database.
  • APOD: Shows the Astronomy Picture of the Day on our Homepage using NASA's apod api
  • Star Ratings: Each tour has a rating at the bottom that logged in users can use.
  • Github Authentication: One way to log in is with Github
  • Profiles: Once a profile is made it is added to our Profiles page where other users can view them
  • NASA Link: Each tour has an external link to NASA's website, particularly their page on that specific tour
